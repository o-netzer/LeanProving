# My Propositional Logic Project in Lean 4

This repository contains my formal proofs for propositional logic as I learn the Lean 4 theorem prover.

## 🚀 Interactive Mode
You can view and run these proofs interactively in your browser without installing anything:

[![Open in GitHub Codespaces](https://github.com/o-netzer/LeanProving)](https://codespaces.new)

## How to use
1. Click the badge above to launch a Codespace.
2. Open any `.lean` file in the file explorer.
3. Wait for the Lean Infoview to start (it will appear on the right).
4. Place your cursor inside a proof to see the goal state!
